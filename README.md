Travel Companion App
The app helps international travellers convert values across three categories: Currency, Fuel, and Temperature.It has a simple UI and is easy to use.

There are three convert Features:  
Currency conversion (USD, AUD, EUR, JPY, GBP)
Fuel efficiency conversion (MPG, KM/L, Liter, Gallon(US), Kilometer)
Temperature conversion (Celsius, Fahrenheit, Kelvin)

Users use the app to enter the values that need to be calculated and select units from the spinners. When the user clicks the convert button, the app calculates based on the conversion logic.

Technologies Used:
Android Studio
Java
XML
